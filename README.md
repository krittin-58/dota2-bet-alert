Dota 2 Bet Alert
===============

A Chrome extension to keep track of active Dota 2 Lounge matches.

Add to Chrome: https://chrome.google.com/webstore/detail/dota-2-bet-alert/ihdninjofmiiephiallglgnmolafiafi

### Requirements
- Node.JS
- gulp

### Setup
- `npm i`

### Build
- `gulp build` or just `gulp`
